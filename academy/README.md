# 项目介绍
## 项目名称
观逸学堂项目
## 项目描述
观逸学堂项目是跨境电商知识学习平台
## 项目功能
- 跨境电商知识学习
- 跨境电商知识分享
- 跨境电商知识问答
## 项目技术栈
- 前端：Vue3 + ElementPlus
- 后端：Flask
- 数据库：MySQL
- 服务器：CentOS
- 域名：www.eyesure.cn
- 备案：www.eyesure.cn
